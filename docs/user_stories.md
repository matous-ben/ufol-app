## User Stories

Aplikace bude mít dva hlavní typy uživatelů — fanouška soutěže a admina. Do budoucna by bylo možné přidat user stories pro registrovaného fanouška s rozšířenými výhodami (sledování oblíbeného týmu, nákupy ve fanshopu apod.), pro první verzi aplikace by to však nedávalo smysl.

---

### Fanoušek (neregistrovaný návštěvník)

| # | User Story |
|---|-----------|
| F-01 | Jako fanoušek chci vidět aktuální tabulku ligy, abych věděl, kdo zrovna vede, a kde se nachází můj oblíbený tým. |
| F-02 | Jako fanoušek si chci zobrazit odehrané zápasy, abych se mohl podívat na výsledky jednotlivých zápasů. |
| F-03 | Jako fanoušek si chci zobrazit naplánované zápasy, abych mohl zjistit, kdy a kde se hraje a mohl utkání případně navštívit. |
| F-04 | Jako fanoušek si chci zobrazit detail zápasu, abych se dozvěděl výsledek a statistiky. |
| F-05 | Jako fanoušek chci vidět soupisky týmů, abych se dozvěděl, kdo nastoupil do zápasu. |
| F-06 | Jako fanoušek chci mít k dispozici přehlednou úvodní stránku, kde uvidím všechno aktuální dění soutěže. |
| F-07 | Jako fanoušek chci mít možnost zobrazit si detail týmu, abych viděl podrobné informace o konkrétním týmu. |

---

### Admin

| # | User Story |
|---|-----------|
| A-01 | Jako admin chci mít možnost se jednoduše a bezpečně přihlásit do interního dashboardu, abych mohl spravovat soutěž. |
| A-02 | Jako admin chci být automaticky odhlášen po určité době neaktivity, aby nedošlo ke zneužití přístupu. |
| A-03 | Jako admin chci mít přehlednou strukturu sekcí (události, týmy, ročníky…), abych mohl rychle spravovat konkrétní část soutěže a měl přístup k archivovaným ročníkům. |
| A-04 | Jako admin chci mít možnost přidat nadcházející zápasy, abych mohl vytvořit budoucí program soutěže. |
| A-05 | Jako admin chci mít možnost spravovat odehrané zápasy, abych mohl zapsat jejich výsledek, sestavy a události. |
| A-06 | Jako admin chci mít možnost spravovat jednotlivé ročníky soutěže, abych mohl vytvářet nadcházející ročníky a archivovat ukončené. |
| A-07 | Jako admin chci spravovat týmy účastnící se soutěže, abych mohl evidovat jejich účast v jednotlivých ročnících. |
| A-08 | Jako admin chci mít možnost spravovat hráče jednotlivých týmů, abych udržoval aktuální informace o aktivních hráčích. |

---

## Plánovaná rozšíření (mimo scope v1)

- Registrovaný fanoušek — sledování oblíbeného týmu, personalizovaný obsah
- Fanshop — nákup merchandise
- Prodej vstupenek na zápasy
- Prezentace partnerů a sponzorů soutěže
- Rozšíření na další univerzitní sporty (florbal, hokej, volejbal)
